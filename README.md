# CS2-Config-Personal-use
这是一个为 **Counter-Strike 2** 设计的综合性 `autoexec.cfg` 配置文件，是我为我自己写的。

它的核心理念是**“模式化键位”**：通过小键盘切换不同的功能模块（练习、Bot控制、娱乐、模式选择），从而复用同一组按键（U, I, O, P, H, J...），在不占用过多键位的情况下实现数十种功能。

## ✨ 主要功能

* **多模式菜单系统**：使用小键盘切换四种不同的操作模式。
* **跑图/练习模式**：一键开启无限弹药、投掷物轨迹、无敌、自动回血。
* **Bot 完全控制**：可以放置 Bot、让 Bot 模仿动作（练习双架）、定身、蹲下。
* **Solo/1v1 模式**：快速设置单挑环境，内置多种枪械局预设（AK, M4, AWP 等）。
* **实用工具**：清除投掷物、切换透视（X-Ray）、显示弹着点、坐标显示。
* **娱乐功能**：一键切换连跳模式（Bhop）、RGB 动态变色 HUD。
* **高级飞行（Noclip）**：支持 5 档飞行速度调节。

## 📥 安装方法

1. 下载或复制本配置代码。
2. 将文件保存为 `autoexec.cfg`。
3. 将文件放入 CS2 的 cfg 目录：
* 通常位于：`\Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg\`


4. 打开 Steam 库 -> 右键 CS2 -> 属性 -> 启动选项。
5. 添加参数：`+exec autoexec.cfg`。
6. 进入游戏后，即可生效。

## 🎮 使用指南 (核心逻辑)

本配置使用 **小键盘 (Numpad)** 作为“模式切换器”。按下对应的小键盘数字，**主键盘区的功能键 (U, I, O, P, H, J, K)** 功能会发生改变。

### 1. 基础键位 (全局通用)

| 按键 | 功能 | 说明 |
| --- | --- | --- |
| **Alt** | 飞行 (Noclip) | 开/关飞行模式 |
| **[** | 降低飞行速度 | 共 5 档 |
| **]** | 提高飞行速度 | 共 5 档 |
| **Y** | 重丢道具 | 重新投掷上一颗道具 |
| **滚轮** | 跳跃 | 连跳必备 |
| **KP_7** | **重置/解绑** | **紧急复位键，解绑所有功能键并显示主菜单** |

---

### 2. 模式菜单 (小键盘控制)

按下以下小键盘按键激活对应模式，控制台会提示当前已激活的模式：

#### 🔢 KP_8：模式选择 (Mode Selection)

*用于切换服务器的游戏规则配置*

* **U**: 载入 **练习模式** (无限弹药, 轨迹, 6颗道具等)
* **I**: 载入 **竞技模式** (正常规则, 关闭作弊)
* **O**: 载入 **Solo 模式** (1v1 专用规则, 需配合控制台指令

#### 🛠️ KP_9：练习功能 (Functions)

*跑图时常用的开关工具*

* **U**: 开关 **无敌模式** (Buddha)
* **I**: 开关 **弹着点显示** (Show Impacts)
* **O**: 开关 **透视模式** (X-Ray / r_aoproxy_show)
* **P**: 开关 **坐标显示** (Show Pos)
* **H**: **清除所有投掷物** (烟/火/闪/雷)
* **J**: 开关 **玩家碰撞体积**

#### 🤖 KP_4：Bot 控制 (Bot Control)

*用于测试身位、双架或瞄准*

* **U**: **添加** 一个 Bot
* **I**: 将 Bot **放置** 在准星瞄准的位置
* **O**: 切换 Bot **移动/静止**
* **P**: 切换 Bot **蹲下/站立**
* **H**: 切换 **模仿模式** (Bot 模仿你的动作 -> 玩家控制 Bot -> 关闭)
* **J**: 重置 Bot 面朝方向 (同向/镜像)
* **K**: 踢出所有 Bot

#### 🌈 KP_5：小程序 (Features)

*娱乐与辅助脚本*

* **U**: 切换 **连跳模式** (关闭 -> 限速连跳 -> 不限速连跳)
* **I**: 切换 **RGB HUD** (WASD 移动时 HUD 变色，再次按下关闭)

### 3. Solo 模式专用指令

在激活 Solo 模式（`KP_8` -> `O`）后，可以在控制台输入以下指令快速开始特定枪械的 1v1 对局：
(指令来自于[Purple-CSGO](https://github.com/Purple-CSGO/CS2-Config-Presets)大佬)

* `ak` / `ak47`: AK 对局
* `m4` / `m4a4`: M4A4 对局
* `m4a1`: M4A1-S 对局
* `awp`: AWP 对局
* `deagle`: 沙鹰对局
* `usp` / `glock`: 手枪局
* *更多支持：galil, famas, aug, sg553, mp9, mac10, xm (喷子) 等...*

## ⚠️ 注意事项

1. **关于 VAC**：本脚本完全由控制台指令组成，**不包含**任何外部程序注入，因此在受 VAC 保护的服务器（如官匹）中使用是安全的。但请注意，脚本中的 `sv_cheats 1` 相关功能（练习、Bot、轨迹等）只能在你自建的房间或开启了作弊权限的服务器中使用。
2. **RGB HUD**：开启 RGB HUD 会绑定 `WASD` 键。如果你在关闭该功能时遇到问题，请按 `KP_7` 重置，或者重启游戏。
3. **加空速**：脚本中包含 `bind mouse_x "+forward;-forward;yaw"`，这用于更快的加空速。如果你不习惯，请在 `autoexec.cfg` 中删除该行。
4. **默认滚轮跳**：脚本中包含 `bind "MWHEELUP" "+jump"; bind "MWHEELDOWN" "+jump""`，这是滚轮跳。如果你不习惯，请在 `autoexec.cfg` 中删除该行。

## 🛠️ 自定义

所有的 `alias` 逻辑都已分块注释，你可以轻松修改默认的飞行速度或练习模式的参数。
